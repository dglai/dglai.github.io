class dgl.nn.pytorch.conv.GatedGraphConv(in_feats, out_feats, n_steps, n_etypes, bias=True)[source]ÔÉÅ

Bases: Module

Gated Graph Convolution layer from Gated Graph Sequence Neural Networks

\[ \begin{align}\begin{aligned}h_{i}^{0} &= [ x_i \| \mathbf{0} ]\\a_{i}^{t} &= \sum_{j\in\mathcal{N}(i)} W_{e_{ij}} h_{j}^{t}\\h_{i}^{t+1} &= \mathrm{GRU}(a_{i}^{t}, h_{i}^{t})\end{aligned}\end{align} \]
Parameters:

  • in_feats (int) ‚Äì Input feature size; i.e, the number of dimensions of \(x_i\).

  • out_feats (int) ‚Äì Output feature size; i.e., the number of dimensions of \(h_i^{(t+1)}\).

  • n_steps (int) ‚Äì Number of recurrent steps; i.e, the \(t\) in the above formula.

  • n_etypes (int) ‚Äì Number of edge types.

  • bias (bool) ‚Äì If True, adds a learnable bias to the output. Default: True.

Example

>>> import dgl
>>> import numpy as np
>>> import torch as th
>>> from dgl.nn import GatedGraphConv
>>>
>>> g = dgl.graph(([0,1,2,3,2,5], [1,2,3,4,0,3]))
>>> feat = th.ones(6, 10)
>>> conv = GatedGraphConv(10, 10, 2, 3)
>>> etype = th.tensor([0,1,2,0,1,2])
>>> res = conv(g, feat, etype)
